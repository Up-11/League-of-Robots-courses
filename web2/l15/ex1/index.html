Переключатель тем

Простой переключатель тем

Эта версия использует только кнопки для переключения между темами. Все циклы, массивы и объекты удалены для максимального упрощения кода.

Текущая тема: Светлая

Как это работает

Каждая кнопка при клике устанавливает соответствующий атрибут data-theme для элемента body, что активирует нужные CSS-переменные.

Выбранная тема сохраняется в localStorage браузера и автоматически применяется при следующем посещении страницы.
